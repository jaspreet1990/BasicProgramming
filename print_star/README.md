# BasicProgramming

How to print * Pattern like :

*
**
***
****
*****

Below is the Solution :

import java.io.*;
 
class myCode
{
    public static void main (String[] args) throws java.lang.Exception
    { 
        
        for(int i=1;i<=5;i++)
        {
            for(int j=1;j<=i;j++)
            {
                System.out.print("*");
            }
             System.out.print("\n");
        }
    }
}
